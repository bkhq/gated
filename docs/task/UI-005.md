# UI-005 目标管理

## 基本信息

- **状态**: review
- **优先级**: P1
- **关联计划**: PLAN-005 Phase 2
- **创建时间**: 2026-03-22

## 描述

实现目标（Target）的完整管理，支持 SSH/HTTP/MySQL/PostgreSQL/Kubernetes 多种类型。

## 任务清单

- [x] 目标列表（搜索 + 按组过滤）
- [x] 创建目标表单（类型切换）
- [x] 目标详情/编辑
- [x] 删除目标
- [x] 角色分配（添加/移除）
- [x] SSH 已知主机密钥查看

## API 端点（10 个）

- `GET /targets` - 目标列表
- `POST /targets` - 创建目标
- `GET /targets/:id` - 目标详情
- `PUT /targets/:id` - 更新目标
- `DELETE /targets/:id` - 删除目标
- `PUT /targets/:id/roles` - 更新角色分配
- `GET /targets/:id/ssh-known-hosts` - SSH 已知主机密钥

## 页面

- 目标列表页
- 创建目标页
- 目标详情页

## 技术要点

- 表单根据目标类型动态切换字段
- react-hook-form + zod 条件验证
- 目标组过滤使用 Select 组件
