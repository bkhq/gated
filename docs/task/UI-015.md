## API 端点（12 个）

- `GET /targets` - 可连接目标列表
- `GET /profile` - 个人资料
- `PUT /profile/password` - 修改密码
- `GET /profile/public-keys` - 公钥列表
- `POST /profile/public-keys` - 添加公钥
- `PUT /profile/public-keys/:id` - 编辑公钥
- `DELETE /profile/public-keys/:id` - 删除公钥
- `GET /profile/otp` - OTP 列表
- `POST /profile/otp` - 添加 OTP
- `DELETE /profile/otp/:id` - 删除 OTP
- `GET /api-tokens` - Token 列表
- `POST /api-tokens` - 创建 Token
- `DELETE /api-tokens/:id` - 删除 Token
