## API 端点（6 个）

- `GET /ssh/known-hosts` - 已知主机列表
- `POST /ssh/known-hosts` - 添加已知主机
- `DELETE /ssh/known-hosts/:id` - 删除已知主机
- `POST /ssh/check-host-key` - 主机密钥检查
- `GET /ssh/own-keys` - 自有公钥列表
- `GET /ssh/own-keys/:id` - 公钥详情
