## API 端点（22 个）

- `GET /users` - 用户列表
- `POST /users` - 创建用户
- `GET /users/:id` - 用户详情
- `PUT /users/:id` - 更新用户
- `DELETE /users/:id` - 删除用户
- `GET /users/:id/credentials/password` - 密码状态
- `PUT /users/:id/credentials/password` - 设置密码
- `GET /users/:id/credentials/public-keys` - 公钥列表
- `POST /users/:id/credentials/public-keys` - 添加公钥
- `PUT /users/:id/credentials/public-keys/:key_id` - 编辑公钥
- `DELETE /users/:id/credentials/public-keys/:key_id` - 删除公钥
- `GET /users/:id/credentials/otp` - OTP 列表
- `POST /users/:id/credentials/otp` - 添加 OTP
- `DELETE /users/:id/credentials/otp/:otp_id` - 删除 OTP
- `GET /users/:id/credentials/certificates` - 证书列表
- `POST /users/:id/credentials/certificates` - 签发证书
- `DELETE /users/:id/credentials/certificates/:cert_id` - 吊销证书
- `GET /users/:id/credentials/sso` - SSO 列表
- `POST /users/:id/credentials/sso` - 添加 SSO
- `PUT /users/:id/credentials/sso/:sso_id` - 编辑 SSO
- `DELETE /users/:id/credentials/sso/:sso_id` - 删除 SSO
- `PUT /users/:id/roles` - 更新角色分配
