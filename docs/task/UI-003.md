# UI-003 布局与导航

## 基本信息

- **状态**: done
- **优先级**: P0
- **关联计划**: PLAN-005 Phase 1
- **创建时间**: 2026-03-22

## 描述

实现 Admin 和 Gateway 两套布局，包含侧边栏导航、面包屑、响应式布局和用户信息下拉菜单。

## 任务清单

- [x] Admin 侧边栏导航（所有资源入口）
- [x] Gateway 顶部导航
- [x] 面包屑组件
- [x] 响应式布局（mobile sidebar drawer）
- [x] 用户信息下拉菜单

## 技术要点

- shadcn/ui Sidebar + Sheet 组件
- React Router Outlet 嵌套布局
- 响应式断点使用 Tailwind CSS
- 导航高亮基于当前路由

## 实现说明

- `AdminLayout`: shadcn SidebarProvider + Sidebar + SidebarInset；侧边栏分 Monitoring / Configuration 两组，带 lucide-react 图标；可折叠（collapsible="icon"）；底部用户菜单使用 DropdownMenu + Avatar；移动端自动切换 Sheet 抽屉（shadcn sidebar 内置）
- `GatewayLayout`: 顶部 Header 含品牌 Logo、目标导航链接、用户 DropdownMenu（Profile / API Tokens / Logout）、ModeToggle；已登录时在 Header 下方显示面包屑条
- `AppBreadcrumb`: 基于 `useMatches` + 路由 `handle.breadcrumbKey` 自动生成面包屑，支持 i18n 插值（如 `Session Detail: {{id}}`）
- `router.tsx`: 为所有子路由添加 `handle: { breadcrumbKey }` 静态元数据
- i18n: 新增 `common.nav.monitoring/configuration`、`common.user.profile/logout`、`gateway.nav.targets/logout`
