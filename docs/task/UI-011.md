## API 端点（8 个）

- `GET /ldap` - LDAP 服务器列表
- `POST /ldap` - 创建 LDAP 服务器
- `GET /ldap/:id` - LDAP 服务器详情
- `PUT /ldap/:id` - 更新 LDAP 服务器
- `DELETE /ldap/:id` - 删除 LDAP 服务器
- `POST /ldap/:id/test` - 连接测试
- `GET /ldap/:id/users` - 用户列表
- `POST /ldap/:id/import` - 批量导入
