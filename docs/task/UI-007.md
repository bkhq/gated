## API 端点（7 个）

- `GET /roles` - 角色列表
- `POST /roles` - 创建角色
- `GET /roles/:id` - 角色详情
- `PUT /roles/:id` - 更新角色
- `DELETE /roles/:id` - 删除角色
- `GET /roles/:id/users` - 关联用户
- `GET /roles/:id/targets` - 关联目标
